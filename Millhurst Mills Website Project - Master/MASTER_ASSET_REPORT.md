## 16. FILES NEEDING RESIZING / CONVERSION

### For Web Use — Convert Canon DSLR Photos to WebP

All 51 Canon DSLR JPGs (7–14 MB) are too large for direct web use. They must be:

1. **Resized** to appropriate dimensions
2. **Converted** to WebP format (or AVIF for cutting-edge performance)
3. **Compressed** for fast page load

| Use Case | Target Width | Target Format | Max File Size |
|----------|-------------|---------------|---------------|
| Full-width hero (desktop) | 2400 px | WebP | 300 KB |
| Full-width hero (mobile) | 800 px | WebP | 120 KB |
| Category card image | 800 px | WebP | 80 KB |
| Gallery image | 1200 px | WebP | 150 KB |
| Thumbnail | 400 px | WebP | 40 KB |

**Conversion command (batch, using cwebp or ImageMagick):**
```bash
# Resize + convert all JPGs to WebP (80% quality)
for f in *.JPG; do
  cwebp -q 80 -resize 2400 0 "$f" -o "webp/${f%.JPG}.webp"
done

# Or using ImageMagick:
for f in *.JPG; do
  convert "$f" -resize 2400x -quality 80 "webp/${f%.JPG}.webp"
done
```

**WhatsApp photos (WA photos):** Already small; convert to WebP as-is for gallery use:
```bash
for f in IMG-20251003*.jpg; do
  cwebp -q 85 "$f" -o "webp/${f%.jpg}.webp"
done
```

**CR2 RAW files:** Do NOT convert for web. Archive in `12_Archive_Old_Unused/RAW_Files/`. They are master originals.
